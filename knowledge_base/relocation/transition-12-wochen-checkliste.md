# 12-Wochen Transition-Checkliste mit Critical Path
*Wissensstand: März 2026*
*Chunk-ID: relocation/transition-12-wochen-checkliste*
*Relevante Agenten: Relocation-Agent, CFO-Agent, Insurance-Agent*

## Critical Path — Dependencies

```
Banking (Woche -12) ──→ MUSS vor Abmeldung stehen
     │
Probeaufenthalt (Woche -8 bis -5) ──→ Go/No-Go Gate
     │
Mietvertrag Zielland (Woche -4) ──→ Basis für alles weitere
     │
Versicherungs-Handoff (Woche -2) ──→ Nahtloser Übergang
     │
Abmeldung DE (Woche -1) ──→ LETZTER Schritt
     │
Ankunft + Registrierung (Woche 0-4) ──→ Zielland-Setup
```

**Goldene Regel:** Abmeldung DE = LETZTER Schritt. Alles andere MUSS vorher stehen.

## Woche -12 (3 Monate vorher)

### Banking [KRITISCHER PFAD]
- [ ] Tolerante DE-Bank identifizieren + Konto eröffnen (DKB/Commerzbank)
- [ ] Wise Multi-Currency-Konto eröffnen
- [ ] Revolut als Backup eröffnen
- [ ] Testüberweisung: DE-Bank → Wise → Revolut (100€)

### Versicherungen
- [ ] Internationale KV beantragen (Gesundheitsprüfung braucht 2-4 Wochen!)
- [ ] PKV: Entscheidung Anwartschaft (klein/groß) oder Kündigung
- [ ] PI/E&O: Internationale Police recherchieren
- [ ] Bestehende Policen auf Auslandsklauseln prüfen (BU, PHV, Rechtsschutz)

### Planung
- [ ] Top-3-Städte finalisiert (aus Länder-Scoring-Matrix)
- [ ] Steuerberater briefen: Wegzugstiming, Steuerjahr-Split
- [ ] Immigration Agent: Visa-Anforderungen für Zielland klären

## Woche -8 bis -5 (Probeaufenthalt — 4 Wochen!)

### Vor Ort leben, NICHT nur besuchen
- [ ] Wohnung mieten (Airbnb oder Kurzzeit-Mietvertrag, Expat-Viertel)
- [ ] Remote Work testen: Internet-Speed, Zeitzone, Coworking
- [ ] Golf-Clubs besuchen (Green Fee, Qualität, TrackMan, Community)
- [ ] CrossFit-Boxen testen (Probetraining, Coach-Qualität, Hyrox-Option)
- [ ] Healthcare: Arztbesuch machen (Qualität, Sprache, Wartezeit testen)
- [ ] Supermärkte, Restaurants, Alltags-Infrastruktur testen
- [ ] Expat-Treffen besuchen (Internations, Facebook-Gruppen, Meetup)
- [ ] Lokale Bank besuchen: "Was brauche ich für Kontoeröffnung?"
- [ ] Lokalen Steuerberater treffen (Residenz-Anforderungen, Kosten)
- [ ] Sicherheitsgefühl evaluieren (Tag + Nacht, verschiedene Viertel)

### Bewertung (am Ende Woche -5)
Bewertungsbogen ausfüllen (aus probe-aufenthalt-plan.md):
- Gewichteter Score ≥3.5/5 → **GO** — Weitermachen
- Score 3.0-3.5 → **CONDITIONAL** — Zweiter Probeaufenthalt in 2-3 Monaten
- Score <3.0 → **NO-GO** — Nächste Kandidaten-Stadt testen, zurück zu Woche -12

## Woche -1 (1 Woche vorher)

### Abmeldung [DER LETZTE SCHRITT]
- [ ] **Einwohnermeldeamt: Abmeldung** → Abmeldebescheinigung erhalten
  - Mitbringen: Personalausweis, Mietvertrag (alt + neu), Bestätigung des Vermieters
  - **DIESES DOKUMENT IST KRITISCH** — für Steuer, Versicherung, Banking
- [ ] Finanzamt informieren (Adressänderung, Steuerjahr-Split)
- [ ] GKV: Abmeldung wirksam ab Wegzugstag
- [ ] Zustellbevollmächtigten benennen (§123 AO) — z.B. Steuerberater oder Eltern
- [ ] Letzte Steuerschätzung/Vorauszahlung (wenn selbständig)

### Abreise vorbereiten
- [ ] Flug gebucht (One-Way oder Flex-Return)
- [ ] **Dokumente-Mappe:** Reisepass, Personalausweis, Geburtsurkunde (Original!), Abmeldebescheinigung, Diplome/Zeugnisse, Arbeitsvertrag/Geschäftsdokumente, Versicherungspolicen, Bankdokumente, Steuerbescheide (letzte 3 Jahre)
- [ ] **Notfall-Nummern:** Dt. Botschaft im Zielland, Int. KV-Hotline, Steuerberater DE + Zielland
